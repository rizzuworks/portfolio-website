To run this project locally on your computer:

1. Clone repository
   ```bash
   git clone https://github.com/rizzuworks/portfolio-website.git
   ```
2. Navigate to project folder
   ```bash
   cd portfolio-website
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start development server
   ```bash
   npm run dev
   ```
